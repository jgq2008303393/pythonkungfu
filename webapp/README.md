To run this app, just install [Flask](http://flask.pocoo.org/) and then:

```
# pip install Flask
$ python webapp.py
```

I suggest you to use nodemon. Install [Node.js](http://nodejs.org/) and then:

```
# npm install nodemon --global
$ nodemon --exec "python" webapp.py
```
